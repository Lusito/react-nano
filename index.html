@react-nano

Tiny, powerful and type-safe React libraries

Type-Safe

All of these libaries are made with TypeScript in mind. You can benefit from this even if you are using pure JavaScript!

Lightweight

Tired of packages having lots of dependencies making your project huge? All included libraries use JavaScript features to stay small.

No external dependencies

Dependencies pose a risk of security vulnerabilities. @react-nano does not have any third-party dependencies (apart from react or similar *peer* dependencies)

# Included Libraries

  • @react-nano/redux
    Lightweight alternative to react-redux.
  • @react-nano/redux-dynamic-modules
    Making redux-dynamic-modules more lightweight by using @react-nano/redux instead of react-redux.
  • @react-nano/router
    Lightweight alternative to react-router.
  • @react-nano/tsrux
    Lightweight alternative to redux-actions, deox, etc.
  • @react-nano/use-event-source
    Hook for using Server-Sent-Events (opens new window).
  • @react-nano/use-fetch
    Hook for using fetch() to GET, POST, etc. requests.
  • @react-nano/use-graphql
    Hook for using GraphQL queries and mutations.
